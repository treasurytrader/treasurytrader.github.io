# 1. 제목 (Headers)
`#`의 개수에 따라 H1부터 H6까지 표현할 수 있습니다. `#` 뒤에는 반드시 한 칸을 띄워야 합니다.

# 가장 큰 제목 (H1)
## 중간 크기 제목 (H2)
### 소제목 (H3)
#### 아주 작은 제목 (H4)

---

# 2. 텍스트 강조 (Emphasis)
글자 모양을 굵게, 기울이게, 또는 취소선을 그어 강조할 수 있습니다.

* **굵게 쓰기**: 글자 앞뒤로 `**` 또는 `__`를 붙입니다. (예: **중요한 내용**)
* *기울여 쓰기*: 글자 앞뒤로 `*` 또는 `_`를 붙입니다. (예: *이탈릭체 강조*)
* ~~취소선~~: 글자 앞뒤로 `~~`를 붙입니다. (예: ~~지워진 텍스트~~)

---

# 3. 목록 (Lists)
순서가 없는 목록과 순서가 있는 목록을 만들 수 있습니다.

### 순서가 없는 목록 (글머리 기호)
`*`, `+`, `-` 중 아무거나 사용해도 동일하게 출력됩니다.
* 사과
* 바나나
  * 하위 목록은 앞에 공백(스페이스바) 2칸을 띄웁니다.
  * 망고

### 순서가 있는 목록 (숫자)
숫자 뒤에 점`.`을 찍어 표현합니다. 숫자를 무작위로 적어도 자동으로 순서대로 정렬됩니다.
1. 첫 번째 할 일
2. 두 번째 할 일
3. 세 번째 할 일

# 4. 링크와 이미지 (Links & Images)
웹사이트 연결 및 이미지를 삽입하는 방법입니다.

* **링크 주소 연결**: `[표시할 이름](URL 주소)` 형식으로 작성합니다.
  * 예시: [구글 사이트 이동](https://google.com)

* **이미지 삽입**: 링크 문법 앞에 느낌표`!`를 붙입니다. `![대체 텍스트](이미지 주소)`
  * 예시: ![노트북 이미지](https://unsplash.com)

# 5. 코드 블록 (Code Blocks)
개발 언어나 프로그래밍 코드를 강조할 때 사용합니다.

### 인라인 코드
문장 중간에 짧은 키워드를 넣을 때는 백틱(`) 1개로 감쌉니다.
* 예시: 터미널에 `git commit` 명령어를 입력하세요.

### 블록 코드 (여러 줄 코드)
백틱(`) 3개로 감싸고, 시작하는 백틱 뒤에 해당 소스코드의 언어 이름을 적으면 문법 강조(Highlighting)가 작동합니다.

```python
def hello_world():
    print("Hello, GitHub Pages!")
```

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```

# 6. 표 (Tables)
수직선`|`과 하이픈`-`을 조합하여 표를 만듭니다. 하이픈 영역에 콜론`:`을 넣어 정렬 방향을 결정합니다.

| 이름 | 역할 (좌측 정렬) | 진행 상황 (중앙 정렬) |
| :--- | :--- | :---: |
| 홍길동 | 메인 개발자 | 완료 |
| 이순신 | UI 디자이너 | 진행 중 |

---

# 7. 인용문 및 체크리스트 (Blockquotes & Task Lists)

### 인용문
오른쪽 꺾쇠괄호 `>`를 문장 맨 앞에 붙여 설명이나 명언을 강조합니다.
> "실패는 성공의 어머니이다." - 에디슨

### 체크리스트 (To-Do List)
대괄호 안에 공백`[ ]` 또는 대문자 X`[x]`를 넣어 할 일 목록을 표현합니다.
- [x] 깃허브 블로그 개설하기
- [x] 오픈 그래프 태그 설정하기
- [ ] 첫 번째 마크다운 글 발행하기
